# CryptoApp
A basic crypto app to show the current value of coins

## Techs

- MVVM
- DI - Hilt
- DataBinding
- Retrofit
- Coroutines
- LiveData
- ViewModel
- Base Classes
- Coil
- Jetpack Navigation
